Sistema de Tarefas – Instruções de Execução
Este projeto é um sistema web simples para gerenciamento de tarefas, desenvolvido utilizando Python e Flask, seguindo conceitos básicos de Programação Orientada a Objetos.

Requisitos para executar
Python 3 instalado
Git
Navegador web

Como executar a aplicação
1. Clonar o repositório
https://github.com/Ricardoo-Vieira/sistema_prat.git

2. Instalar o Flask
pip install flask --break-system-packages

Ou, dependendo da máquina:
pip install flask

3. Executar o sistema
python app.py

4. Acessar a aplicação
Abra no navegador:
http://127.0.0.1:5000
